Hay varias maneras de trabajar formularios en Angular pero principalmente tenemos los formularios reactivos y los formularios con Template
la diferencia es que los formularios por template, la logica y el codigo esta en el HTML, cosa que con los reactivos tenemos mejor separacion
de logica y mayor control

Entre las carpetas de la estructura de directorio que tenemos estan:
  - auth -> Esto es para manejar el ingreso aunque todabia no nos vamos a conectar a un backend
  - shared -> Aqui estaran los modulos compartidos entre las diferentes piezas de los diferentes modulos
  - utils -> Como los formularios reactivos reutilizan mucho codigo, asi que solo copiamos y pegamos lo que requieramos

Problemas que tenemos al inicio
  - Los dos checkbox se pueden seleccionar y solo se debe de poder seleccionar uno
  - Recuperar el valor del select si queremos o no recibir notificacion
  - Que sea obligatorio el check de los terminos y condiciones
  - Que al precionar el boton de guardar no se active la propagacion del formulario

Campos de lista deplegables anidadas
  Queremos que al elegir una opcion de una lista, esto influya en las demas opciones que mostraran una segunda lista desplegable
  ademas al seleccionar una opcion de esta segunda lista, afectara a las opciones que muestre una tercera lista desplegable

  Vamos a configurar la aplicacion para hacer peticiones HTTP, traer la data de un Api que consumamos
  en los endpoints de la API de paises, ya por medio de parametros mandamos en la URL, obtenemos la informacion requerida para mostrar en los selectores
  y sus datos anidados, ademas de consumir otros endpoints para mostrar la informacion correcta sin abreviaciones
    - https://restcountries.com/

  Ahora en este punto vamos a hacer que el tercer selector muestre los resultados segun la opcion elegida en el segundo selector, en el caso del tercer selector
  se mostraria los paises con los que comparte frontera el pais elegido en el selector dos, pero tenemos el problema que la API nos esta regresando el codigo de cada pais
  y es mejor mostrar al usuario el nombre del pais por lo que tenemos que camiar eso
